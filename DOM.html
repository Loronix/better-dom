staticDOM.emmet(template, varMap){String}

better-dom.js, line 234
Parse emmet-like template and return resulting HTML string
Name Type Description
template String input EmmetString
varMap Object | Array optional key/value map of variables
See:
  • https://github.com/chemerisuk/better-dom/wiki/Microtemplating
  • http://docs.emmet.io/cheat-sheet/
Returns:
resulting HTML string
Example
DOM.emmet("a");                                    // => '<a></a>'
DOM.emmet("ul>li*2");                              // => '<ul><li></li><li></li></ul>'
DOM.emmet("b>`hello {user}`", {user: "world"});    // => '<b>hello world</b>'
DOM.emmet("i.{0}+span", ["icon"]);                 // => '<i class="icon"></i><span></span>'
DOM.emmet("i.{a}>span#{b}", {a: "foo", b: "bar"}); // => '<i class="foo"><span id="bar"></span></i>'
